# 🧠 Voice-Based Parkinson's Disease Classification
Machine learning project developed for the Machine Learning course (2025–2026) at the University of Padova.

## **Overview**
This project investigates whether voice biomarkers can distinguish individuals with Parkinson's disease from healthy subjects. Since Parkinson's disease can affect speech patterns, voice measurements may provide a low-cost, non-invasive screening approach.

**Dataset**: UCI Machine Learning Repository

**Methods**
- Exploratory data analysis
- Feature preprocessing and duplicate removal
- Stratified train/validation/test split
- Feature scaling with StandardScaler
- Model comparison:
  - Logistic Regression
  - Random Forest
- Cross-validation and learning curves
- Performance evaluation using F1-score, ROC-AUC, precision, recall, and confusion matrices
